Products
Get all products:
GET /api/v1/products

Get a specific product by ID:
GET /api/v1/products/:id

Create a new product:
POST /api/v1/products
Body: { "product_name": "", "content": "", "price": "", "isNew": "", "image": "", "variants_color": "", "variants_size": "", "variants_quantity": "", "category_id": "", "merchant_id": "" }

Update a product:
PATCH /api/v1/products/:id
Body: { "product_name": "", "content": "", "price": "", "isNew": "", "image": "", "variants_color": "", "variants_size": "", "variants_quantity": "", "category_id": "", "merchant_id": "" }

Delete a product:
DELETE /api/v1/products/:id

Search Products:
GET /api/v1/products/search
Query Parameters:
  - q (string): Search query term
  - category_id (optional, integer): Filter by category
  - price_min (optional, number): Minimum price filter
  - price_max (optional, number): Maximum price filter
  - ratings(optional, number): Filter by ratings
